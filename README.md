# Student Budget & Expense Tracker

A simple Python based project that helps students manage their monthly expenses, track spending categories, and stay within their budget.  
This project is created as part of an academic coursework requirement and focuses on clean modular code, basic file based storage, and an easy command line interface that beginners can understand and run.

---

## Features

- Set or update monthly budget  
- Add expenses with amount, category, date & description  
- View all recorded expenses  
- Get summary including:
  - Total spent  
  - Remaining budget  
  - Percentage of budget used  
  - Category wise totals  
- JSON based data storage
- Fully modular Python code (5 files)
- Beginner friendly and easy to run

---

## Project Structure

main.py          → main menu & program flow  
storage.py       → load/save JSON data  
budget.py        → budget management functions  
expenses.py      → add/view expenses  
reports.py       → summary & category analytics  
data.json        → auto created database file  
README.md        → documentation  
statement.md     → problem statement & project scope
